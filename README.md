# fsb-docs

Documentation site for [TG-FileStreamBot](https://github.com/EverythingSuckz/TG-FileStreamBot), built with [VitePress](https://vitepress.dev).

**Live site:** [filestream.bot](https://filestream.bot)

## Development

```bash
bun install
bun run docs:dev
```

## Build

```bash
bun run docs:build
```
